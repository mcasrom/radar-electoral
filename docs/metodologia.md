# Metodología Sistema Multicapa 2026

## 1. Datos
- Fuentes: CIS, GAD3, Sigma Dos, 40dB
- Provincias: 52, con escaños asignados por población
- Pesos: sqrt(muestra) / (1 + antigüedad/30) * factor de fiabilidad

## 2. Procesamiento
- Media ponderada por partido y provincia
- Simulación Monte Carlo: 2000 iteraciones
- Algoritmo D’Hondt por provincia
- Suma nacional y cálculo de bloques

## 3. Indicadores
- Fragmentación parlamentaria: Laakso-Taagepera
- Volatilidad temporal: desviación estándar de PP
- Radar estratégico: factores externos (vivienda, energía, defensa, inflación, territorio)

## 4. Narrativa OSINT
- Automática, interpretando bloques, riesgos y fragmentación
