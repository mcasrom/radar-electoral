## 🔹 Descripción

`España Vota 2026` es un sistema de **inteligencia geopolítica y electoral** que:

- Analiza la intención de voto semanal y mensual en España.  
- Genera proyecciones de hemiciclo usando **D’Hondt** por provincia.  
- Integra factores de tensión socioeconómica (vivienda, energía, inflación, defensa, territorio).  
- Combina fuentes oficiales y simuladas con un **coeficiente de veracidad** para ponderar resultados.  
- Produce dashboards visuales y tablas analíticas en **Streamlit**, con exportación a CSV y gráficos interactivos.
